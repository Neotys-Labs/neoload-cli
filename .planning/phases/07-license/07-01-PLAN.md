# Phase 7: License Management

v4 command for license and lease operations.

## Tasks

### 1. v4-license (`v4_license.py`)

Subcommands:
- `get` — GET /v4/license
- `install` — POST /v4/license
- `leases-ls` — GET /v4/license/leases (workspaceId optional query param)
- `leases-create` — POST /v4/license/leases (requires workspaceId in body)
- `leases-get` — GET /v4/license/leases/{leaseIdentifier} (downloads lease file)
- `activation-request` — POST /v4/license/activation-requests
- `deactivation-request` — POST /v4/license/deactivation-requests
- `forced-release` — POST /v4/license/forced-releases
- `release` — POST /v4/license/releases

Options: `--file` for JSON body input, `--workspace` for lease operations.
